# 🏡 House Price Prediction using Machine Learning

## 📖 Project Overview

This project predicts house prices using the **California Housing Dataset** from Scikit-learn. It demonstrates the complete machine learning workflow, including data preprocessing, feature scaling, model training, performance evaluation, and model comparison.

Three regression models are trained and evaluated to determine the best-performing model.

---

## 🎯 Objectives

* Load and explore the California Housing Dataset
* Apply feature scaling using StandardScaler
* Train multiple regression models
* Compare model performance using RMSE and R² Score
* Visualize model performance
* Predict house prices accurately

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook
* VS Code

---

## 🤖 Machine Learning Models

* Linear Regression
* Ridge Regression
* Decision Tree Regressor

---

## 📊 Evaluation Metrics

The models are evaluated using:

* Root Mean Squared Error (RMSE)
* R² Score

The best model is selected based on the **lowest RMSE** and **highest R² Score**.
